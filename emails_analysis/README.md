#Email activity analysis by month

## Project Description
This SQL query is designed to analyze email activity broken down by month and specific accounts.

### Key Features:
* **Data Aggregation:** Calculates the number of messages for each month.
* **Window Functions:** Uses window functions to calculate percentages.
* **Time Tracking:** Identifies first and last sending dates.

### Technologies:
* **Google BigQuery SQL**
* `SUM() OVER`, `MIN() OVER`, `MAX() OVER`
